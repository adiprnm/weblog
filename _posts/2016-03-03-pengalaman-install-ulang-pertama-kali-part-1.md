Rencananya kemarin saya mau janjian dengan teman saya (teman SMA yang satu kampus dengan saya, hanya saja beda jurusan), ketemuan di Mesjid Raya Unpad. Mau ngambil laptopnya yang katanya mau diinstall ulang. Tapi tidak jadi karena sudah terlalu malam, dan sayanya juga males kalau harus berjalan dari asrama ke Mesjid Raya Unpad yang menghabiskan waktu sekitar 30 menit. Capek kakak!

Jadi ceritanya kira-kira begini. Beberapa hari sebelumnya dia mengeluh karena laptop miliknya itu bermasalah saat start-up. Saya pun memberi saran kepadanya sesuai dengan pengetahuan yang saya miliki. Tapi tetap saja, masalahnya tidak terselesaikan.

“Tempat install ulang laptop di Jatinangor di mana ya?” tanyanya.

Bermodalkan pengetahuan install ulang yang saya dapatkan dari pengamatan terhadap teman-teman saya yang jago dalam hal beginian, flashdisk 16 Gigabyte milik teman saya yang sudah dibuat menjadi bootable flashdisk, dan koneksi internet di kampus yang cukup ngebut, saya menawarkan diri untuk meng-install ulang laptopnya. “Tapi bayar ya, he he he,” ucap saya kemudian.

“Iya, tapi harga ‘dulur’ ya…” balasnya. Dulur itu artinya saudara. Saya pun mematok harga 20 ribu, memang bisa dibilang sangat murah untuk biaya install ulang sebuah laptop yang biasanya berkisar antara 30-50 ribu. Biarlah, mengingat ini adalah pertama kalinya saya melakukan proses install ulang laptop tanpa bantuan dari orang lain, dalam artian meminta bantuan kepada teman-teman saya, saya tidak ingin mengambil untung yang terlalu besar.

Baru tadi siang, kami janjian untuk ketemuan di Mesjid Raya Unpad. Kali ini jadi. Teman saya itu datang bersama temannya, kemudian menyerahkan tas laptop berwarna pink, khas perempuan sekali. Kami pun berbincang-bincang sejenak, membicarakan tinggi badan saya yang semakin tinggi saja di matanya, dan beberapa hal kecil lainnya. Kemudian dia pamit, saya juga pamit.

Sesampainya di asrama saya langsung download ‘barang-barang’ yang diperlukan, seperti Windows 7 Ultimate 32bit (karena laptopnya ini tipe laptop ‘jadul’, kelihatan dari body-nya yang tebel buanget), Microsoft Office Enterprise 2007, WinRAR, PowerISO (buat ektrak file .iso), dsb. Sementara untuk K-Lite Code Pack (buat video player), dan Foxit Reader (buat baca e-book), saya dapatkan installer-nya dari laptop saya.

Awalnya sempat bingung pada saat pertama kali masuk menu awal instalasi, tepatnya ketika harus memilih partisi yang mana, karena ada 3 partisi: partisi 1 ukutannya hanya 100 MB, partisi 2 ukurannya 68 GB, dan partisi 3 ukurannya 130 GB. Saya pun memutuskan untuk memilih partisi 2, dan alhamdulillah ternyata pilihan saya tepat.

Terus sempat kaget juga, karena setelah proses instalasi selesai, komputernya melakukan restart. Bukannya melanjutkan ke tahap yang selanjutnya, prosesnya malah kembali ke proses instalasi semula. Akhirnya saya install lagi dari awal, dengan harapan bisa lanjut ke proses selanjunya. Tapi masih tetap saja. Dan ternyata setelah diselidiki, saya mengubah prioritas booting-nya dengan flashdisk saya, dan ketika proses hendak akan me-restart, flashdisk-nya tidak saya cabut, dan saya pikir ini adalah penyebab utamanya.

*Bersambung, dilanjut besok karena ngantuk.*
