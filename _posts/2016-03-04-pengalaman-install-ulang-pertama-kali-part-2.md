Post ini adalah lanjutan dari [post saya sebelumnya](/pengalaman-install-ulang-pertama-kali-part-1), karena ngantuk, he he.

Setelah tahu apa penyebabnya, proses selanjutnya pun berjalan lancar. Tapi setelah berhasil meng-install, ada beberapa kendala yang saya hadapi. Yang pertama, proses check disk pada saat start-up. Pada awalnya saya biarkan saja sampai prosesnya selesai. Maka kemudian saya tinggal tidur selama 1 jam. Kejanggalannya, prosesnya itu terhenti pada suatu proses, namanya lupa. Semacam hang begitulah. Saya pun mencoba bersabar sedikit lagi, tapi tetap saja prosesnya tidak bergerak. Maka saya pun langsung mematikan laptop secara paksa, dan pada kesempatan selanjutnya saya memencet sebuah tombol sembarang untuk melewati proses check disk ini.

Sampai di titik ini, proses install ulang bisa dibilang berhasil.  Tapi karena sebelumnya saya salah langkah, di Local Disk C terdapat 2 folder windows.old yang berbeda, dan tentunya ini menghabiskan ruang Local Disk C saja. Untungnya bisa dihapus, dengan cara menggunakan Disk Cleanup.

Selanjutnya saya hanya perlu menginstall aplikasi yang dikira perlu, seperti driver-driver, Microsoft Office, E-book Reader, WinRAR, dan sebagainya. Untungnya lagi nih, koneksi internet di sini cukup cepat, jadi saya tidak mengalami kendala serius dalam hal men-download software-software yang diperlukan itu. Kendalanya hanya di Microsoft Office, sempat salah install, saya malah menginstall Microsoft Project. Salah baca kali ya, he he. Dan masih di Microsoft Office, agak sulit mencari Microsoft Office Professional Plus 2007, yang saya dapatkan malah yang versi Enterprise-nya. Karena tidak mau ambil pusing, saya pun menginstall yang versi Enterprise saja.

Setelah semuanya beres, hanya satu yang tersisa: proses check disk otomatis saat start up. Saya pun mencari cara untuk mematikannya, ada banyak di Google. Pake cara cmd ternyata tidak berhasil. Kemudian saya menemukan cara yang pada akhrinya berhasil, yaitu dengan mengatur pengaturannya di regedit.

Yah, begitulah pengalaman install ulang saya yang pertama kali. Dan tadi siang habis jumatan saya telah mengembalikan laptop itu kepada sang pemiliknya.
