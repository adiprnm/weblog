Salah satu alasan kenapa saya lebih senang nulis di blog ketimbang di media sosial, adalah karena blog itu sangat customizable. Kita bebas menambahkan, mengubah, atau mengurangi apapun di blog kita, tanpa terikat oleh pihak manapun.

Hasilnya: halaman di blog ini bisa dibilang cukup banyak, dan nggak expect akan cukup banyak juga sih. Setiap ada sekumpulan hal menarik yang menarik yang ingin saya bagikan, saya biasanya buat halaman baru.

Misal, saya membuat halaman khusus untuk tulisan-tulisan seputar teknologi di halaman [Teknologia](/teknologia). Atau, membuat halaman khusus untuk [newsletter](/newsletter) mingguan (walaupun sekarang belum ada plan untuk lanjut lagi sih). Ada juga halaman berisi sekumpulan [Tulisan Menarik](/tulisan-menarik).

Halaman [Status](/status) pun pada awalnya nggak direncanakan sama sekali. Ide status terbaru di bawah halaman baru muncul setelah mengunjungi salah satu website, dan di bagian bawah halaman ada status terbaru sang penulis. Karena cukup menarik, saya buat juga lah di blog ini.

Itu pun di versi pertama memunculkan status terbaru aja, persis seperti website yang menginspirasi tersebut. Halaman Status baru saya buat ketika saya ingin melihat status-status yang udah lampau di satu halaman.

Selain halaman random, saya juga bikin beberapa boring tools. Ada tools untuk [generate link chat WA](/tools/chat-wa-link-generator). Ada tools untuk [share undangan pernikahan](/tools/share-undangan). Ada juga tools untuk melihat [tanggal merah terdekat](/tools/tanggal-merah) dan [menghitung jumlah waktu](/tools/time-calculator).

Semua tools ini berawal dari kebutuhan saya pribadi, yang kemudian saya bagikan secara cuma-cuma. Harapan saya, tools ini bisa sedikit-banyak membantu pengguna dalam menyelesaikan permasalahannya.

Tech stack-nya pun nggak macam-macam: hanya vanilla JavaScript dan bantuan PicoCSS supaya nggak repot ngurusin styling lagi.

Pada akhirnya, aktivitas menambahkan, mengedit, dan mengurangi konten blog yang berlangsung terus menerus ini secara tidak langsung menciptakan sebuah personal hub yang bisa dieksplorasi oleh pengunjung.

Saya percaya, manusia itu nggak terbatas hanya pada titel profesionalnya saja: Senior Software Engineer, AI Engineer, Graphic Designer, dan titel-titel mentereng lainnya. Banyak hal menarik yang bisa dibagikan diluar dari kegiatan profesional: hobi, traveling, atau opini terhadap peristiwa yang lagi happening.

Jadi, what's stopping you from making your own custom personal hub?
