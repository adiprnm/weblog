Quick update: saya akhirnya merealisasikan project yang saya maksud di [post ini](/the-programmers-problem).

Sebenarnya ini nggak direncanakan. Kebetulan saya lagi nyoba-nyoba Open Code. Lalu terbesitlah ide untuk merealisasikan project ini.

Waktu itu saya lagi malas ngoding, jadi saya serahkan semuanya ke Open Code. Bisa dibilang 99% buatan AI. Saya hanya edit-edit sedikit aja.

Saya ingin tes seberapa bagusnya Open Code untuk vibe coding aplikasi Rails. Hasilnya ternyata sangat memuaskan. Desainnya seperti yang saya harapkan: minimalis. Fitur-fitur utamanya juga udah ada

- search post berdasarkan nama atau base url
- filter post berdasarkan kategori
- submit post baru
- halaman admin

![BlogFeed](/assets/images/2025/Screenshot_20251111_142342_Chrome.webp)
*Tampilan halaman beranda BlogFeed*

Mekanismenya, user submit post yang ingin ditampilkan di feed. Nggak perlu login untuk ini. Sebagai gantinya, setiap post yang di-submit akan saya moderasi terlebih dahulu untuk mengantisipasi spam dan post bermuatan negatif.

Jadi, ayo submit blog post mu di sini: [https://blogfeed.adipurnm.com](https://blogfeed.adipurnm.com). Sementara masih pakai subdomain, tapi kalau suatu hari semakin banyak orang yang submit, saya akan pertimbangkan beli domain baru, hihi.
