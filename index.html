Saya Adi. Seorang programmer Ruby on Rails yang hobi menulis dan menikmati hidup slow living di desa bersama istri dan 1 anak.

Saat ini, saya bekerja secara remote di Happy5 sebagai Backend Engineer.

Saya hobi menulis sejak tahun 2012. Dimulai dari ketertarikan terhadap teknologi. Kemudian, bergelut di dunia web design sebagai script kiddies dan copas konten blog orang lain. Hingga akhirnya menyadari bahwa yang membuat website hidup bukanlah desainnya, tapi tulisan-tulisan otentik di dalamnya.

Sepanjang perjalanan, saya menulis di berbagai platform. Hingga akhirnya memutuskan untuk fokus menulis di website pribadi supaya lebih sederhana dalam banyak aspek.

Saya menulis macam-macam, mulai dari tulisan teknis, hingga tulisan nonteknis. As the website name implies: log. Dan sebagaimana log dalam suatu aplikasi: ada access log, error log, atau warning log.

Saya juga menjalankan Kanyaah Undangan Digital dan Bikin Aja Dulu. List project yang pernah/sedang saya kerjakan saya selengkapnya ada di halaman Project.

Jangan lupa isi Buku Tamu ya! Biar saya bisa silaturahmi ke blog kamu juga, hehe.
