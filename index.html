Saya Adi. Seorang programmer Ruby on Rails yang hobi menulis dan menikmati hidup slow living di desa. Bekerja secara remote di Happy5 sebagai Backend Engineer.

Saya hobi menulis sejak tahun 2012. Sepanjang perjalanan, saya menulis di berbagai platform. Hingga akhirnya memutuskan untuk fokus menulis di website pribadi supaya lebih sederhana dan mudah dikustomisasi.

Saya menulis macam-macam, mulai dari tulisan teknis, hingga tulisan nonteknis. As the website name implies: log. Dan sebagaimana log dalam suatu aplikasi: ada access log, error log, atau warning log.

Saya juga menjalankan Kanyaah Undangan Digital dan Bikin Aja Dulu. List project yang pernah/sedang saya kerjakan saya selengkapnya ada di halaman Project.

Jangan lupa isi Buku Tamu ya! Biar saya bisa silaturahmi ke blog kamu juga, hehe.
